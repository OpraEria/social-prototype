# Social prototype for validated learning
This repo is a bare-bones version for testing core assumptions about our social app solution.

## Prereqs
- Azure sql (or equivalent) provisioned
- Env vars containing db creds + secrets
- Node.js (lts) and any package manager (npm / yarn / pnpm / bun)

## Setup
Install deps:
```bash
npm install
```

## Run PWA

```bash
npm run dev
```
